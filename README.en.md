# 宿舍管理系统

#### Description
基于SSM开发的宿舍管理系统，前端采用的是jsp、jquery。功能模块分：登录、首页、学生管理、班级管理、宿舍管理、卫生管理、访客管理、管理员管理。代码规范整洁，简单运行，适用于毕业设计以及计算机课程设计。
